Add extra features to Attesa WordPress theme

== Description ==

<strong>Attesa Extra</strong> add extra features to <a href="https://attesawp.com" target="_blank">Attesa Theme</a> like demo import, custom templates, widgets, elementor widgets and metaboxes to edit the structure of the website for each post/page.<br />
Attesa Extra <strong>requires <a href="https://attesawp.com" target="_blank">Attesa theme</a></strong> to be installed.

== Installation ==

1. Upload the folder 'attesa-extra' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= I installed the plugin but it does not work =
Make sure you have <a href="https://attesawp.com" target="_blank">Attesa Theme</a> installed because Attesa Extra is a plugin designed to work exclusively with Attesa theme
